# Screenshot Directories

| Directory | Purpose | Tracked in Git? |
|-----------|---------|----------------|
| `baseline/` | Reference screenshots | Yes |
| `current/` | Screenshots during test runs | No |
| `diff/` | Diff images on failure | No |

## Naming
`<page-name>-<viewport>.png`

## Viewports
- Desktop: 1440x900
- Mobile: 375x812
